繰り返し

処理を何度も繰り返す

  • 処理を繰り返す
    • while, do…while
    • for
  • 繰り返しを抜ける
    • break;
  • 繰り返しを1回だけ飛ばす
    • continue;

単純な繰り返し

while

while文で指定された条件に合致している間は処理を繰り返す。

while ( 条件 ) {
    条件が真値(true)の間、繰り返す処理
}

10回繰り返す

while文では、繰り返しの条件となるものが必ず必要となる。

int count = 1;
while ( count <= 10 ) {
    繰り返す処理をここに書きます。
    count += 1;
}

do…while

条件の検査が中括弧を閉じるタイミングで行われるwhile

中括弧内の処理は、条件に関わらず一度は実行される。

do {
    処理を実行後、条件が真値の場合には何度も繰り返す
} while ( 条件 ); //セミコロンを忘れずに付けましょう！

10回繰り返す

int count = 1;
do {
    繰り返す処理 (countの初期値が11だった場合でも処理は実行されます)
    count += 1;
} while ( count <= 10 );

繰り返し用の変数を別で用意するのめんどくないですか？

for

whileループの繰り返し条件用の数値型変数を簡略化して書けるループ

初期化された変数が、繰り返し都度変化します。変化した値が繰り返し条件に合致しなくなった場合に繰り返しを抜けます。

for (変数の初期化; 繰り返し条件; 繰り返し時の変化) {
    繰り返す処理
}

10回繰り返す

for (int count = 1; count <= 10; count++) {
    繰り返す処理
}

繰り返し処理中に例外的な処理を行う

繰り返し中に別の条件でループを抜けたい

break

ループ処理にかぎらず、ブロックから抜け出すことができる命令。

for文を例にすると…

for (変数の初期化; 繰り返し条件; 繰り返し時の変化) {
    繰り返す処理
    if ( 別の条件 ) break;
}

一度だけ処理を飛ばして、次の繰り返しを行う

continue

ループ処理中に、continueが実行されると、それ以降の処理は1回の繰り返し処理時のみ飛ばして次の繰り返し処理が実行される。

for (変数の初期化; 繰り返し条件; 繰り返し時の変化) {
    繰り返す処理
    if ( 別の条件 ) continue;
    別の条件に合致する場合、実行されない処理
}

特殊な繰り返し処理

次に紹介する繰り返し処理は、現時点で使用することができない繰り返し処理です。これらは後ほどの講義で解説いたします。

  1. イテレータ(Iterator)
    • 配列やコレクションのような順次的なデータ構造のオブジェクトに対して、反復して処理を繰り返しさせるためのメソッドを実装したもの。
    • 「次の要素があるか」という条件分岐と、「次の要素を取り出す」処理を併せて使用される。
    • 反復子という訳語が用いられる場合もあるがこの呼び方は一般的ではない。
  2. 拡張for
    • イテレータと同じく配列やコレクションのような順次的なデータ構造のオブジェクトに対して、要素を一つずつ取り出し、すべての要素を順次処理していく際に用いられるfor構文のこと。
    • 配列のインデックスを取得する必要がない場合に直感的な要素の取り出しが可能となる繰り返し処理。
